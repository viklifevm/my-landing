# Telegram-бот для лендинга ВМ.Код

## Задача
Собрать Telegram-бота, который принимает заявки с формы лендинга и пересылает их владельцу.

## Как должно работать
1. Пользователь заполняет форму на лендинге и жмёт «Разобрать мой бизнес»
2. Данные формы (имя, контакт, описание задачи) отправляются боту
3. Бот пересылает заявку в личку Виктории (@VMuzychenko8)

## Поля формы (что передавать боту)
- Имя
- Email или Telegram
- Описание: где теряете деньги / что нужно автоматизировать

## Стек
- Python (python-telegram-bot) или Node.js
- Webhook или polling
- Без базы данных — только пересылка сообщений

## Шаги реализации
1. Создать бота через @BotFather, получить токен
2. Написать скрипт бота (принимает POST-запрос с данными формы, отправляет сообщение)
3. Задеплоить на хостинг (Railway, Render или VPS)
4. В index.html заменить `e.preventDefault()` на реальный fetch к боту
5. Протестировать
